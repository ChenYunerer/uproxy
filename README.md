# uproxy
系统代理快捷设置工具

###### add proxy
##### uproxy -a (-http http://127.0.0.1:1087 -http http://127.0.0.1:1087)
###### remove proxy
##### uproxy -r
###### show proxy config
##### uproxy -s

###### -http http代理地址 (default "http://127.0.0.1:1087")
###### -http  https代理地址 (default "http://127.0.0.1:1087")
